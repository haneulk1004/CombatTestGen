# CombatTestGen v1.0 Release Notes

**출시일**: 2026-02-06

## 🎉 주요 기능

- **Unity C# 테스트 자동 생성**: 전투 파라미터만 입력하면 Test Runner용 코드가 완성됩니다.
- **강력한 엣지 케이스 엔진**: 0 데미지, 1 HP 생존, 오버킬(Overkill) 상황을 자동으로 테스트에 추가합니다.
- **직관적인 GUI**: 비개발자도 쉽게 사용할 수 있는 Windows UI를 제공합니다.
- **CLI & 배치 모드**: CI/CD 파이프라인에 통합할 수 있는 강력한 CLI를 지원합니다.

## 🛠 수정 사항

- 초기 릴리즈입니다.

## 📦 설치 방법

1. [Releases 페이지](#)에서 `CombatTestGen.zip` 다운로드
2. 압축 해제 후 `CombatTestGen.exe` 실행
