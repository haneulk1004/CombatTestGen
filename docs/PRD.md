게임 전투 컨텐츠 테스트 케이스 자동 생성 프로그램(이하 "CombatTestGen")은 개발자가 전투 시스템 파라미터를 입력하면 Unity C#/Python 테스트 코드를 즉시 생성해주는 독립 실행 도구입니다. Python 기반 CLI/GUI 앱으로 배포되며, 엣지 케이스까지 자동 포함해 QA 시간을 80% 단축합니다.

## 목적과 목표

- **비즈니스 목표**: 게임 개발 생산성 향상, 테스트 케이스 작성 시간 1시간 → 5분으로 단축.
- **사용자 목표**: 비프로그래머 PM/QA도 쉽게 전투 테스트 코드 생성.
- **KPI**: 생성 테스트 케이스 100개당 버그 발견률 30%↑, 사용자 만족도 4.5/5.0.

## 기능 요구사항

## 핵심 기능

1. **입력 인터페이스**
    - CLI: `combattestgen --damage 50 --hp 100 --scenarios normal,critical`
    - GUI: HP/데미지/시나리오 입력 폼
2. **테스트 케이스 생성**
    
    `text[Test]
    public void TestNormalAttack_50Damage_100HP()
    {
        // HP 100 → 50 확인
    }`
    
3. **엣지 케이스 자동 생성**
    - 0 데미지, 최대 HP, 치명타 999%, 적 사망 등 10+ 케이스.

## 사용자 여정

`text1. 앱 실행 → 2. 전투 파라미터 입력(HP, 데미지, 속도)
          → 3. 시나리오 선택(기본/치명/회피)
          → 4. "생성" 클릭 → 5. CombatTests.cs 자동 저장
          → 6. Unity Test Runner에서 검증`
